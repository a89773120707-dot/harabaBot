# Настройка Cron (Расписание)

## Текущее расписание: каждые 15 минут

Чтобы установить:
```bash
crontab -e
```

Добавьте строку:
```cron
*/15 * * * * cd /opt/haraba-mini && ./run_pipeline.sh
```

## Как изменить интервал

В Cron используется формат: `минута час день месяц день_недели`

| Интервал | Cron выражение | Пример |
|----------|----------------|--------|
| **Каждые 15 минут** | `*/15 * * * *` | Запуск в :00, :15, :30, :45 |
| **Каждые 30 минут** | `*/30 * * * *` | Запуск в :00, :30 |
| **Каждый час** | `0 * * * *` | Запуск в :00 |
| **Каждые 4 часа** | `0 */4 * * *` | Запуск в 00:00, 04:00, 08:00... |
| **Каждый день в 9 утра** | `0 9 * * *` | Запуск в 09:00 |

**Пример изменения на каждые 4 часа:**
```cron
0 */4 * * * cd /opt/haraba-mini && ./run_pipeline.sh
```

**Пример изменения на каждый час:**
```cron
0 * * * * cd /opt/haraba-mini && ./run_pipeline.sh
```
