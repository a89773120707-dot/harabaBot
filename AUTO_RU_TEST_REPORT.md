## Технические наблюдения

### ✅ Что работает
1. **CSS селектор** `div.ListingCars__universalSnippetWrapper` — находит все 37 карточек
2. **ext_id** из URL — 100% coverage
3. **Цена** — 100% coverage, точные числа
4. **Год** — 100% coverage
5. **URL** — 100% coverage
6. **Регион** — 97% coverage

### ⚠️ Что требует доработки
1. **Пробег** — regex захватывает лишние цифры. Нужно точнее парсить: `(\d[\d\s]*\d)\s*км`
2. **Бренды** — расширить список: MG, Chery, FAW, Geely, Haval, Dongfeng, Peugeot
3. **Popup "Модели-близнецы"** — закрывается по Escape, но лучше кликать по X
4. **Карточки с overlay** — "Ещё 6 фото", "Только на Авто.ру" перекрывают название авто

###  Подход к парсингу
- **НЕ работает:** `page.content()` → сырой HTML без JS-рендера
- **Работает:** `page.evaluate()` → извлечение из DOM после JS-рендера
- **Работает:** `card.inner_text()` через Playwright locator
- **Не авторизация не нужна** — все данные доступны без входа
